Anúncio de Símbolos

  • Autor: Noelia Ruiz Martínez.
  • Download stable version (compatible with NVDA 2019.3 and beyond)
  • Download development version (compatible with NVDA 2019.3 and beyond)

Este extra permite ouvir os símbolos digitados (caracteres não alfanuméricos ou invisíveis), mesmo quando a opção \"Anunciar caracteres\" está desactivada no NVDA.

Este extra é baseado no antigo complemento ReportSymbols, desenvolvido pelo mesmo autor. deve desinstalá-lo para usar esta versão.

Menu preferências

  • Configurações do anúncio de Símbolos: Permite definir preferências para ler os símbolos digitados.

Nota: O comando para abrir esta caixa de diálogo pode ser atribuído no menu NVDA, Preferências, Definir comandos, na categoria Configuração.
